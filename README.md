# MyCodeChavez Portfolio 🚀

Este es el portafolio personal de **MyCodeChavez**, un desarrollador web creativo enfocado en experiencias interactivas y modernas.

## 🌟 Características

- **Fondo Interactivo 3D**: Partículas animadas utilizando Three.js que reaccionan al movimiento del mouse.
- **Efecto de Resplandor del Cursor**: Un halo de luz sigue al usuario por toda la interfaz.
- **Proyectos Dinámicos**: Integración con la API de GitHub para mostrar repositorios actualizados automáticamente.
- **Diseño Glassmorphism**: Interfaz moderna con efectos de desenfoque y bordes sutiles.
- **GitFlow**: Estructura de ramas profesional para un desarrollo organizado.

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3 (Variables, Animaciones, Flexbox/Grid).
- **JavaScript**: Vanilla JS, Three.js (para efectos 3D).
- **Herramientas**: Git, GitHub API, FontAwesome.

## ✅ Definition of Done (DoD)

Para que una tarea o mejora se considere completada en este proyecto, debe cumplir con los siguientes criterios:

1.  **Funcionalidad**: La mejora debe funcionar correctamente en los navegadores modernos (Chrome, Firefox, Edge).
2.  **Responsividad**: La interfaz debe adaptarse a dispositivos móviles y tablets.
3.  **Rendimiento**: Las animaciones 3D no deben degradar significativamente el rendimiento del sitio.
4.  **Código Limpio**: El código debe seguir las convenciones establecidas (variables CSS, comentarios claros).
5.  **Flujo de Git**:
    - Las mejoras se desarrollan en ramas `feature/`.
    - Se integran a `develop` tras ser verificadas.
    - `main` solo contiene versiones estables listas para producción.
6.  **Experiencia de Usuario (UX)**: La interactividad debe ser fluida y no entorpecer la navegación.
